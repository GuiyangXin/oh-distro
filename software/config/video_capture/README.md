# Video Capture

To enable video capture on a machine add the following to your ``~/.bashrc`` and also set the location where to save the logs:

```
export OH_VIDEO_CAPTURE=1
export OH_VIDEO_CAPTURE_LOCATION=~/logs/video-logs
```
